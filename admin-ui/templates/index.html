What this controls

When startup config sync is enabled, the VEIN container rewrites startup-managed values like ports, max players, and HTTP settings from environment variables into `Game.ini` and `Engine.ini` every time the container starts.

If you disable it, the INI files become the source of truth for those fields, but users who only manage the server with Compose will no longer get automatic config updates on startup.
